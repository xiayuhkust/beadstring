# 界面风格研究计划——神圣性 × 清晰表达 × 和缓氛围

> 起因（2026-07-22）：两轮对比度修补后仍有用户反映"看不清"。用户提问：是否有必要换一个 UI 风格（神性差一点，但清晰可爱，如"小日常"的色调与卡通风格）。

## 一、问题诊断：看不清的根源不是"暗"，是"一切都想融进背景"

现在的设计语言是"轻声细语"：低透明度底色、细边框、大字距、微弱提示——作为**艺术品**它成立，作为**每天要用的工具**它让眼睛做功太多。两轮修补都是在"轻声细语"的框架内调音量，天花板有限。

但"小日常式卡通"是另一个极端：它解决清晰，代价是把旷野换成了贴纸。串成动画、呼吸入场、沙丘颗粒这些"神圣时刻"在卡通语境里会失效。

**判断：不整体换风格，做"第二主题"实验。** 神圣性不等于黑暗——修道院的清晨也是神圣的。

## 二、可以兼顾的证据（参照系）

| 产品 | 借鉴点 |
|---|---|
| Headspace | 冥想产品做到了明亮+圆润+安静：大色块、暖橙、插画极简。证明"和缓"不需要暗色 |
| Lectio 365 / Pray.com | 灵修类深色 UI，但按钮全部实体化、对比度达标——"暗而清晰"的做法 |
| 小日常 | 借它的**结构**不借它的皮：有限的今日清单、完成即翻面、月历脚印（结构已在吸收：今日完成态/串珠历） |
| YouVersion / 微读圣经 | 读经工具的字号/行距/日夜切换惯例，读经页应向工具标准看齐 |
| 纸艺/低多边形插画（abeto 系） | "可爱"的正确打开方式：几何羊、纸山、蓖麻树，而非表情包 |

## 三、三个价值各自的抓手（设计原则）

- **神圣性**：垂直留白、serif 标题、串成/呼吸的仪式动效、金色只给"珠与线"（稀缺才神圣）
- **清晰**：全部可点元素实体化（有底、有边、有厚度）；正文对比度 ≥ WCAG AA（4.5:1）；提示文字不低于 0.65 透明度；状态变化用颜色+图形双通道
- **和缓**：暖色域（沙、米、赭、金）、大圆角、柔和插画、无红色无惊叹号无角标

## 四、方案：双主题「旷野之夜 / 旷野之晨」

- **夜（现有）**：保留，继续按第三节原则打磨——它是气质担当
- **晨（新做）**：暖纸底（米白 #f5ead6 系）+ 深棕字（#3a2a1e 系）+ 琥珀金点缀；卡片是"纸上的纸"（微阴影）；沙丘剪影换成晨光淡彩。清晰度天然高一个量级，老人友好
- 切换：读经页先试（日夜切换本就是读经软件惯例），默认跟随时段或手动

## 五、执行步骤

1. **令牌化**：把颜色/边框/圆角抽成 CSS 变量（一处改，全站变）——一切实验的前提
2. **晨主题打样**：只做读经页一页，部署后丢进伙伴群 A/B（"哪张截图你愿意每天打开？"）
3. 按反馈决定：晨主题铺全站 / 只留读经页 / 放弃
4. 插画层（可选，后期）：串成页的低多边形小场景（每卷一景的雏形）

## 六、验收判据

- 中老年用户手机上不放大能读全部按钮
- 截图发到微信群里"好看、想点"
- 串成动画在新主题下仍有"成了"的重量感
- 两主题共用一套布局代码，只差令牌
